**CoroUtil** - це базовий бібліотечний [мод для Minecraft](/mods/minecraft), який містить загальний код, на якому ґрунтуються інші моди автора *Corosus*. Він включає покращену навігацію для мобів, виправлення логіки дверей та парканів, а також забезпечує платформу для модів, що використовують цей "двигун". 

### Основні можливості бібліотеки

- **Надійна бібліотека коду для модів** - дозволяє створювати моди швидше та ефективніше.

- **Покращений path-finding (маршрутизація мобів)** та логіка об’єктів (дверей, парканів) в порівнянні з ванільним.

- **Сумісність з різними лоадерами**: Forge, Fabric, NeoForge.

- **Працює в ролі "основи"** - багато модів автора вимагають саме його для роботи.
